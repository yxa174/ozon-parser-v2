# Ozon Parser v2

Парсер товаров с Ozon.ru с автоматическим сбором ссылок из категорий.

## Структура

- `main.py` — парсинг товаров (цена, рейтинг, характеристики, доставка и т.д.)
- `parser_list.py` — сбор ссылок на товары из категорий Ozon
- `urls.txt` — список ссылок на товары
- `urls_catalogs.txt` — список категорий для парсинга ссылок

## Установка

```bash
pip install undetected-chromedriver selenium
```

## Использование

### 1. Сбор ссылок на товары из категорий

Отредактируй `urls_catalogs.txt` — добавь ссылки на категории Ozon (по одной на строку).

```bash
python parser_list.py
```

Результат сохранится в `urls.txt`.

### 2. Парсинг товаров

```bash
python main.py
```

Скрипт откроет ozon.ru, подождёт загрузку, затем перейдёт по ссылкам из `urls.txt` и сохранит данные в JSON-файл `ozon_products_YYYYMMDD_HHMMSS.json`.

## Данные товара

- Название, цена, старая цена, скидка
- Бренд, продавец, артикул, цвет
- Рейтинг, количество отзывов, вопросов
- Категории (хлебные крошки)
- Картинки (основная + галерея)
- Характеристики
- Варианты (цвет/размер)
- Доставка (сроки, способы, цены)
- Распродажа (остатки)
- Время парсинга

## Требования

- Python 3.8+
- Chrome / Chromium
- undetected-chromedriver
